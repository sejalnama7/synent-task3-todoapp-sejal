 ✅ To-Do List Web Application

📌 Project Overview
This is a dynamic To-Do List web application built using HTML, CSS, and JavaScript. It allows users to manage daily tasks efficiently.

🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Persistent data using localStorage

🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

📂 Project Structure
- todo.html → Structure of the app
- todo.css → Styling
- todo.js → Functionality and logic

⚙️ Functionality
- Tasks are added dynamically
- Tasks can be marked as complete
- Tasks remain saved even after page refresh using localStorage


📖 Learning Outcomes
- JavaScript DOM manipulation
- Event handling
- LocalStorage usage
- Building interactive web apps

📌 Conclusion
This project helped me understand JavaScript logic and how to build interactive and functional web applications.
